# Configuracoes do meu ambiente Hyprland

Repositorio com minhas configuracoes pessoais para um setup em Hyprland.

## Componentes

### fastfetch
- Modificacao da logo (Cachy + PNG pessoal).

### kitty
- Ajustes simples de terminal.

### mako
- Personalizacao das notificacoes.

### wofi
- Personalizacao visual do lancador de aplicativos.

### hypr
- Configuracoes do Hyprland.
- Script de menu de energia com opcoes de shutdown, reboot, lockscreen e logout.
- Personalizacao do Hyprlock.
- Configuracao de suspensao com Hypridle.

### waybar
- Personalizacao da Waybar com scripts e atalhos para paineis de Wi-Fi, Bluetooth e audio.

Layout:
- Desktops 1-7
- Aplicativos abertos
- Data e hora
- Trays: bateria, Wi-Fi, Bluetooth, audio, brilho, memoria RAM, temperatura e nucleos de CPU

Imagem da Waybar:

![Waybar](./waybar.png)

## Arquivos principais

- `brave-flags.conf`: flags para mitigar problemas de aceleracao de hardware no Hyprland com NVIDIA.
- `historico_comandos.txt`: historico de comandos usados na instalacao de pacotes e configuracao do sistema.
- `Pacotes_instalados.txt`: lista/organizacao de pacotes e comandos de configuracao filtrados.
